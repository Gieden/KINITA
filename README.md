# KINITA - Retail & Inventory Management

**KINITA** is a high-fidelity retail management application featuring a modern "Antigravity" glassmorphic aesthetic.

## 🚀 How to Run

1.  **Install Dependencies** (if you haven't already):
    ```bash
    npm install
    ```
2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
3.  **Open in Browser**:
    Navigate to [http://localhost:5173](http://localhost:5173).

## ✨ Features
- **Dashboard**: Real-time sales metrics and revenue tracking.
- **Glassmorphism**: Sleek, translucent UI components with depth effects.
- **Responsive Navigation**: Interactive sidebar and header.

## 🛠 Tech Stack
- **Framework**: React + Vite
- **Styling**: Vanilla CSS (CSS Variables, Glassmorphism)
- **Icons**: Lucide React
